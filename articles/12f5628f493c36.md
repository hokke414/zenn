今回は「ノートパソコンのすゝめ」と題して、新大学生に向けたノートPCの選び方についてまとめました。ほかの方の役に立てば幸いです。


## 1. ノートPCはここを見ろ！


ノートPCに限らずですが、パソコンのスペックの良さを決めるものは、大きく分けて4つと言われています。

 - CPU(プロセッサ)
 - GPU(グラフィックボード)
 - RAM(メモリ)
 - SSD/HDD(ストレージ)

よくわからない略語がたくさんありますね。順にみていきます。


### (1) CPU(プロセッサ)


CPUは別名 **「中央処理装置」** とも呼ばれ、パソコンの中枢、つまり頭脳のとしてほぼすべての計算を行います。足りなくなると、パソコンがカクついたり、動作が重くなってしまいます。

CPUにも種類があります。Windowsでは「Intel」,「Ryzen」が主流で、Macには「Aシリーズ」や「Mシリーズ」(Apple CPUとまとめられることがある)が存在します。


:::message

CPUの性能を比べたいときは、「PassMarkベンチマークスコア」を利用します。
これはCPUの性能を

 - 整数演算
 - 浮動小数点演算
 - 暗号処理
 - 圧縮
 - マルチスレッド処理

を考慮してつけられたスコアで、もちろん高いほど性能が高く、悪いほど性能が悪いとされます。

大学等で要求された最低要件CPUのベンチマークスコアを調べてメモしておくと、CPUの名前がよくわからなくてもスコアを比べるだけでパソコン選びが少し楽になるでしょう。
:::


### (2) GPU(グラフィックボード)


GPUは、PCで
- 3D描画
- 動画編集
- AI処理
- 映像・画像描写

を高速で行うのに必要なパーツです。

主にデスクトップPCに「Geforce」や「Radeon」等の外部GPUが搭載されることが多いですが、 **一部のデスクトップPCや大部分のノートPCでは内蔵されたGPUで十分なことが多いので、あまり重要ではないでしょう。**

:::message
GPUも先ほど同様ベンチマークスコアで性能の比較ができます。
:::


### (3) RAM(メモリ)

RAMは別名 **「主記憶装置」** とも呼ばれています。よく作業中の「机の広さ」に例えられており、容量が大きいほど一度に多くのアプリを快適に処理することができます。

こちらはCPUやGPUと違って容量の単位「GB(ギガバイト)」で表すため、理解に苦しむことはないでしょう。

個人の感想を含みますがRAM容量の目安として、以下の表を参考にしてみてください。文系・理系で大きく用途が分かれると思うので注意してください。

| RAM容量 | 最低限できること |
| ---- | ---- |
| 4GB | Webの閲覧 |
| 4~8GB | 文書ファイルの編集・閲覧 |
| 8~16GB | 動画・画像編集 |
| 16~32GB | ゲーム・AI・VRの使用 |


### (4) SSD/HDD(ストレージ)


SSD/HDD(ストレージ)はスマホと同様、写真や動画、ファイル等を保存できる容量を指します。

大学生はレポートの作成、実験データ等の作成・入力はパソコンで行うと思うので、容量が小さいと保存しきれなくなり外部USBメモリなどを追加で買うことになってしまうので十分余裕がありそうなノートPCを選びましょう。

:::message
個人的には256~512GBあれば十分だと思います。
:::


これらがノートPCのスペックに関する評価方法ですが、他にもいくつか注目してみると後々うれしいポイントがあります。


### (5) 重さ


ノートPCは家だけでなく大学やカフェでも使うことになるでしょう。持ち運ぶときにノートPCとはいえ重いと移動が大変になってしまいます。

評価は以下の通りです(個人の意見)。

| 重量        | 評価 |
| --------- | -- |
| 〜1.2kg    | 神  |
| 1.3〜1.5kg | 良い |
| 1.6kg〜    | 重い |


### (6) バッテリー連続駆動時間


人間だれでもやるべきことを忘れることがあります。ノートPCの充電を忘れて使えない...ってなったりすると大変です。なのでそんな心配をなくすためにも連続駆動時間をチェックするのをお勧めします。

個人的には10時間程度持つものがあれば安心かと思います。またバッテリー劣化に応じてバッテリー交換ができるノートPCもあるのでおススメです。
